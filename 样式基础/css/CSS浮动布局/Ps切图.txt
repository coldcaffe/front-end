Ps基本操作:
    1.文件 -> 打开:打开需要测量的图片
    2.Ctrl + R:打开标尺,或者视图->标尺
    3.右击标尺,将其中单位改为像素
    4.Ctrl+加号(+)可以放大视图,Ctrl+减号(-)缩小视图
    5.按住空格键,鼠标可变成小手,拖动PS视图
    6.用选区拖动,可以测量大小

常见图片格式:
    1.jpg(jpeg)图像格式:对色彩信息保留较好,高清且颜色较多,产品类的图片经常用jpg格式的;
    2.gif图像格式:gif最多存储256色,通常用来表示简单图形文字,但可以保存透明背景和动画效果,经常用于图片小动画效果
    3.png:一种新兴的网络图形格式,结合了gif和jpeg的优点,存储形式丰富,能保存透明背景,想要切成背景透明图片,选择png格式;
    4.psd:photoshop的专用格式,可以存放图层,通道,遮罩等多种设计稿,前端可以直接从上面复制文字,获得图片,还可以测量大小和背景

ps切图方式:
    1.图层切图
        右击图层->快速导出为png
        或合并图层再导出:
            (1)选中需要图层:图层菜单->合并图层(ctrl + e)
            (2)右击->快速导出为png

    2.切片切图
        (1)利用切片工具手动划出选中图片
        (2)文件菜单->导出->存储为web设备所用格式->选择我们要的图片格式->存储(导出选中图片)

    3.ps插件切图等
        ps中Cutterman插件,能够自动将你需要的图层进行输出,以替代传统的手工用切片工具切片后导出web格式操作的繁琐流程
        注意:cutterman要求ps必须是完整版,不能是绿色版的(https://www.cutterman.cn/zh/cutterman)
