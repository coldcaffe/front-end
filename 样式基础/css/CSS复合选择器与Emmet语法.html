复合选择器总结:

1.后代选择器:选择后代元素,较多使用 格式 '标签 子孙标签 {}'
2.子代选择器:选择最近一级元素,较少使用 格式 '标签 > 子标签 {}'
3.并集选择器:选择某些相同样式的元素,较多使用 格式 '标签1,标签2,... {}'
4.链接伪类选择器:跟链接相关不同选择,较多使用, 重点记住a{}和a:hover在实际开发中的常见组合使用
5.:focus选择器:用于选择获得光标的表单,较少使用,记住input:focus的写法
