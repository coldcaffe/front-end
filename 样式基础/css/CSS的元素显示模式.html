CSS的元素显示模式总结

1.块级元素: 一行只能放一个块级元素,可以设置宽度高度,宽度默认为容器100%,容器级可以包含任意标签(h1 ~ H6,p,div,ul,ol,li等)
2.行内元素: 一行可放多个行内元素,不可以直接设置宽高,宽度默认为其本身内容的宽度,可容纳文本或其他行内元素(a,strong,b,em,i,del,s,ins,u,span等)
3.行内块元素: 一行放多个行内块元素,可以设置宽高,宽度默认为其本身内容宽度(如img,input,td)
